# Elevvo Machine Learning Internship Projects

A comprehensive collection of machine learning projects completed during the Elevvo internship program. This repository contains 8 different machine learning tasks organized across three difficulty levels, showcasing various ML techniques from basic regression to advanced deep learning.

## Repository Structure

```
Elevvo-Internship/
├── internship folder task level 1/          # Beginner Level Tasks
│   ├── Task1_Student_Score_Prediction/     # Linear Regression
│   └── Task2_Customer_Segmentation/        # K-Means Clustering
├── internship folder task level 2/          # Intermediate Level Tasks  
│   ├── Task3_Forest_Cover_Classification/   # Multi-class Classification
│   ├── Task4_Loan_Approval_Prediction/     # Imbalanced Data Handling
│   └── Task5_Movie_Recommendation_System/  # Collaborative Filtering
├── internship folder task level 3/          # Advanced Level Tasks
│   ├── Task6_Music_Genre_Classification/   # Audio Processing & Deep Learning
│   ├── Task7_Sales_Forecasting/            # Time Series Analysis
│   └── Task8_Traffic_Sign_Recognition/     # Computer Vision & CNNs
├── Machine_Learning_Tasks/                  # Additional ML Tasks
│   ├── Medium/                             # Medium Difficulty Tasks
│   └── Hard/                               # Hard Difficulty Tasks
├── requirements.txt                         # Python Dependencies
├── .gitignore                              # Git Ignore File
└── README.md                               # This File
```

## Project Overview

### Level 1: Beginner Tasks
- **Task 1: Student Score Prediction** - Linear regression for predicting exam scores
- **Task 2: Customer Segmentation** - K-means clustering for customer analysis

### Level 2: Intermediate Tasks  
- **Task 3: Forest Cover Classification** - Multi-class classification with multiple algorithms
- **Task 4: Loan Approval Prediction** - Binary classification with imbalanced data handling
- **Task 5: Movie Recommendation System** - Collaborative filtering and matrix factorization

### Level 3: Advanced Tasks
- **Task 6: Music Genre Classification** - Audio feature extraction and deep learning
- **Task 7: Sales Forecasting** - Time series analysis and forecasting
- **Task 8: Traffic Sign Recognition** - Computer vision with CNNs and transfer learning

## Installation and Setup

### Prerequisites
- Python 3.7 or higher
- pip package manager

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/Elevvo-Internship.git
cd Elevvo-Internship

# Install dependencies
pip install -r requirements.txt
```

### Running Individual Tasks
```bash
# Level 1 Tasks
python "internship folder task level 1/Task1_Student_Score_Prediction/student_score_prediction.py"
python "internship folder task level 1/Task2_Customer_Segmentation/customer_segmentation.py"

# Level 2 Tasks
python "internship folder task level 2/Task3_Forest_Cover_Classification/forest_cover_classification.py"
```
